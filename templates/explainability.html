How to Read This

+ Positive impact means: when this feature has a higher value in a given prediction, the model tends to be correct. A strong positive-impact feature is a reliable signal.

− Negative impact means: when this feature has a higher value, the model tends to be wrong. This can indicate a feature the model over-weights, or a situation type that confounds the prediction.

Method: For each completed prediction, we compare feature values between correct calls and incorrect calls. Impact = mean(correct) − mean(wrong). Minimum {{ [attribution.get('feature_impacts', {}) | length, 5] | min }} samples required per feature.
